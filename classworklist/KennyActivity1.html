ET 710 - Web Technology: Building and Maintaining Websites

Course Description

This course focuses on frontend, browser-based user experience. Students will create websites using current web design patterns and publish them to the public internet. Topics include: Building a webpage with HTML...CSS, Java Script, industry standard web frameworks, responsive web design, data input forms, source and revision control.
